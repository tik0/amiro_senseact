====== Building the RRL based Navigation ======

This demo contains the navigation based on the Robot Robot Localization (RRL). For more detailed information, please check the [[sandbox:driveOnRRL:start|driveOnRRL documentation]].

===== Run the Demo =====

  - ''. ./build.sh .''
  - ''./copyPrograms.sh <ip address>''
  - ''./run.sh'' on the AMiRo
  - ''./stop.sh'' on the AMiRo for stopping

One could use ''rsb-loggercpp0.11 / --style monitor'' to inspect the communication

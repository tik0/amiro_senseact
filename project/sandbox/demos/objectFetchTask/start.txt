====== Building the Object Fetch Task ======

===== Run the Demo =====

  - ''. ./build.sh .''
  - ''./copyPrograms.sh <ip address>''
  - ''./run.sh'' on the AMiRo
  - ''./stop.sh'' on the AMiRo for stopping

One could use ''rsb-loggercpp0.11 / --style monitor'' to inspect the communication

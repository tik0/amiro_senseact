====== Building the Drive On Demand ======

This demo contains the program for driving on demand. Via RSB there can be given commands for driving to a given position or for setting the lights. For detailed application and procedure information, please check the [[sandbox:driveOnDemand:start|driveOnDemand documentation]].

===== Command structure =====

There can be given specific commands for driving to a position and for setting the lights' color and light type. These commands are defined in the [[sandbox:driveOnDemand:start|driveOnDemand documentation]].

For sending commands, the prepared scripts in the directory 'Command' can be used:

  - ''./stop.sh'' for sending the stop command
  - ''./color.sh'' for sending the color command (change color file to your specific command)

===== Run the Demo =====

  - ''. ./build.sh .''
  - ''./copyPrograms.sh <ip address>''
  - ''./run.sh'' on the AMiRo
  - ''./stop.sh'' on the AMiRo for stopping

One could use ''rsb-loggercpp0.11 / --style monitor'' to inspect the communication

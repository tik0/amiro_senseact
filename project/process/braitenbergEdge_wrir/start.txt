====== braitenbergEdge_wrir ======

WARNING: This project is deprecated and is just for development tests! The edges are not detected correctly!

This is the basic braitenberg behavior. In contrast to the [[process:braitenbergEdge:start|normal braitenbergEdge]] it reads the sensor values seperates them in obstacle and ground values by itself.

^ Parameter ^ Description ^
| --loadOffsets, -l | Loads offset from the file "irConfig.conf". |

For additional parameter information type ''braitenbergEdge_wrir -h''.

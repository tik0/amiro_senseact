====== AMiRo DataRecording demo ======

This demo builds all sensing programs.
You have to run rsBag on a remote Host for recording, which is listening to the AMiRo RSB scopes.

===== Run the demo =====

  - ''. ./build.sh .''
  - ''./copyPrograms.sh''
  - ''./run.sh'' on the AMiRo
  - ''./stop.sh'' on the AMiRo for stopping

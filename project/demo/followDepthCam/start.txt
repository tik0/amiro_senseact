====== Building Follow by Depth Camera ======

This demo contains the program for the following procedure by the depth camera. Via RSB the commands for starting and stopping can be given. For more detailed information, please have a look into the [[process:actingBehavior:follow_DepthCam:start|follow procedure documentation]].

===== Command structure =====

For sending commands (starting and stopping the following), the prepared scripts in the directory 'Command' can be used:

  - ''./stop.sh'' for sending the stop command
  - ''./init.sh'' for sending the the start command

===== Run the Demo =====

  - ''. ./build.sh .''
  - ''./copyPrograms.sh <ip address>''
  - ''./run.sh'' on the AMiRo
  - ''./stop.sh'' on the AMiRo for stopping

One could use ''rsb-loggercpp0.11 / --style monitor'' to inspect the communication
